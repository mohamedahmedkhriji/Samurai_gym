# SAMURAI GYM

Premium cinematic landing page for SAMURAI GYM, a fitness brand in Tunisia.

## Features

- Dark luxury fitness landing page
- Responsive React single-page layout
- Animated hero section with brand artwork
- Services, About, Coaches, Pricing, Gallery, and Contact sections
- Local brand assets, coach images, icons, and fonts
- GSAP scroll reveals and Framer Motion interactions
- WhatsApp-ready contact form
- Google Maps embed

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- GSAP
- Framer Motion
- Lucide React
- React Hook Form

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Node Version

Vite may warn if Node is below its preferred patch version. Use Node `20.19+` or `22.12+` for the cleanest setup.
